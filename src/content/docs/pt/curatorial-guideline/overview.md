---

O sistema NAHPU foi projetado para agilizar o processo de curadoria e gerenciamento de coleções de história natural, desde a coleta em campo até a curadoria no museu. Esta diretriz fornecerá algumas abordagens para este processo. Em última análise, caberá à instituição decidir a melhor forma de implementar o sistema para atender às suas necessidades.
