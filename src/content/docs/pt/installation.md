## Status de desenvolvimento do NAHPU

O NAHPU está atualmente em versões candidatas a lançamento. Isso significa que o aplicativo ainda está em teste beta e pode conter bugs ou recursos incompletos, mas é estável o suficiente para uso geral. Estamos nos aproximando do lançamento final nas lojas de aplicativos oficiais.

## Instruções de instalação

### Windows

Requisitos: Windows 10 ou posterior.

Para instalar o aplicativo, siga estas etapas:

1. Baixe a versão mais recente do aplicativo no [GitHub](https://github.com/hhandika/nahpu/releases/latest/download/nahpu-Windows-x86_64.zip).
2. Extraia o arquivo ZIP baixado para uma pasta em seu computador.
3. Para facilitar o acesso, crie um atalho para o arquivo `nahpu.exe` em sua área de trabalho ou barra de tarefas.
4. Abra o aplicativo clicando duas vezes no arquivo `nahpu.exe` ou no atalho que você criou.
5. Se você encontrar um aviso sobre um editor não reconhecido, clique em "Mais informações" e depois em "Executar mesmo assim" para prosseguir com a instalação.

### Android

Requisitos: Android 8.0 (Oreo) ou posterior.

Para instalar o aplicativo, siga estas etapas:

1. Baixe a versão mais recente do aplicativo no [GitHub](https://github.com/nahpu/nahpu/releases/latest/download/nahpu-Android.apk).

2. Abra o arquivo APK baixado em seu dispositivo Android.
3. Se solicitado, ative a instalação de fontes desconhecidas nas configurações do seu dispositivo.
4. Siga as instruções na tela para concluir a instalação.
5. Uma vez instalado, você pode encontrar o aplicativo na sua gaveta de aplicativos ou na tela inicial.

## Linux

Requisitos: Ubuntu 22.04+ ou outras distribuições com GLIBC 2.34+

Para instalar o aplicativo, siga estas etapas:

1. Baixe a versão mais recente do aplicativo no [GitHub](https://github.com/nahpu/nahpu/releases/latest/download/nahpu-Linux-x86_64.tar.gz).
2. Extraia o arquivo `nahpu-Linux-x86_64.tar.gz` baixado para uma pasta em seu computador.
3. Execute o executável nahpu.

## Dispositivos Apple (iOS/iPadOS/macOS)

Requisitos:

- iOS 15.6 ou posterior
- iPadOS 15.6 ou posterior
- macOS 11.5 ou posterior

A Apple exige que o testador beta use o Apple [TestFlight](https://developer.apple.com/testflight/).

Suporte a dispositivos:

- A lista dos dispositivos macOS mais antigos suportados pode ser encontrada no [Suporte da Apple](https://support.apple.com/en-us/102861).
- O suporte a dispositivos para iOS e iPadOS pode ser encontrado no [iosref](https://iosref.com/ios).

Para instalar o aplicativo, siga estas etapas:

1. Baixe e instale o aplicativo TestFlight na App Store em seu dispositivo iOS ou iPadOS.
2. Abra este link em seu dispositivo com o aplicativo TestFlight instalado: [instalar NAHPU](https://testflight.apple.com/join/DRG6yUfb).
