O NAHPU é um projeto gratuito e de código aberto. Se você gosta do NAHPU e deseja apoiar o seu desenvolvimento, pode fazê-lo doando para o projeto. Todas as doações são usadas para pagar os custos de registro do aplicativo, dispositivos de teste e outras despesas.
