## Introdução

O NAHPU permite que você automatize seu fluxo de trabalho para gerar um formulário de entrada para o banco de dados do museu.

Temos um código inicial escrito em R. Estamos planejando adicionar Python em um futuro próximo.

Verifique o código no [GitHub](https://github.com/hhandika/nahpu-scripts)
