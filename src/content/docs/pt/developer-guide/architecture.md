## Visão Geral

NAHPU é um aplicativo multiplataforma construído com [Flutter](https://flutter.dev/), um framework de alto desempenho para a construção de aplicativos multiplataforma. O Flutter usa a linguagem de programação Dart, que é otimizada para a construção de interfaces de usuário. Uma abordagem com o Flutter simplifica o processo de desenvolvimento e permite que os desenvolvedores construam aplicativos para várias plataformas usando uma única base de código. No backend, o NAHPU usa [SQLite](https://www.sqlite.org/index.html) para armazenamento local. Desde o início de 2024, estamos trabalhando para adicionar mais código Rust à base de código do NAHPU. Nosso principal objetivo é reescrever as partes mais lentas da base de código em Rust. Rust é uma linguagem de programação de sistemas que se concentra em desempenho, confiabilidade e segurança. Ao integrar o Rust com o Flutter, nosso objetivo é melhorar o desempenho do aplicativo e fornecer uma experiência de usuário mais robusta.
