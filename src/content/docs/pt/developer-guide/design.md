## Visão Geral

Este documento descreve os princípios de design que orientam o desenvolvimento do NAHPU. Os princípios destinam-se a ajudar os desenvolvedores a tomar decisões que se alinhem com os objetivos e valores do projeto.

## Sistema de Design

O NAHPU usa o sistema [Material Design](https://material.io/design) como base para seu design. O Material Design é uma linguagem de design desenvolvida pelo Google e está bem integrada ao Flutter, o framework usado para construir o NAHPU. Sempre que possível, seguimos as diretrizes do Material Design para garantir uma experiência de usuário consistente e intuitiva em todas as plataformas. Essa abordagem simplifica o processo de design e ajuda os desenvolvedores a se concentrarem na construção de recursos, em vez de projetar componentes de UI do zero.
