## Visão geral

Você pode contribuir para o projeto escrevendo a documentação. Algumas maneiras de contribuir:

- **Atualizar a documentação existente**: Você pode ajudar a manter a documentação atualizada, atualizando as páginas existentes com novas informações ou corrigindo erros.
- **Adicionar nova documentação**: Você pode ajudar a expandir a documentação adicionando novas páginas ou seções que abordam tópicos não abordados atualmente.
- **Traduzir a documentação**: Você pode ajudar a tornar a documentação mais acessível, traduzindo-a para outros idiomas (em breve).

## Passos simples para contribuir

Esta opção é para contribuições rápidas. Se você quiser fazer alterações mais significativas, consulte a seção "Passos avançados para contribuir" abaixo.

### Membros da equipe

Na parte inferior de cada página do site, você encontrará um link "Editar esta página". Clique neste link para abrir a página no GitHub. Você pode então fazer alterações diretamente no GitHub и enviar uma solicitação pull.

### Colaboradores externos

Se você não for um membro da equipe, ainda poderá contribuir para o projeto. Veja como:

1. Faça um fork do repositório.
2. Faça alterações na documentação.
3. Envie uma solicitação pull.

## Passos avançados para contribuir

Este passo permite mais flexibilidade. A documentação é escrita em Markdown e construída usando o Docusaurus. Você pode aprender mais sobre o Docusaurus visitando o [site oficial](https://docusaurus.io/).
