## Status pengembangan NAHPU

NAHPU saat ini dalam versi kandidat rilis. Ini berarti aplikasi masih dalam pengujian beta dan mungkin berisi bug atau fitur yang tidak lengkap, tetapi cukup stabil untuk penggunaan umum. Kami mendekati rilis final di toko aplikasi resmi.

## Petunjuk instalasi

### Windows

Persyaratan: Windows 10 atau lebih baru.

Untuk menginstal aplikasi, ikuti langkah-langkah berikut:

1. Unduh versi terbaru aplikasi dari [GitHub](https://github.com/hhandika/nahpu/releases/latest/download/nahpu-Windows-x86_64.zip).
2. Ekstrak file ZIP yang diunduh ke folder di komputer Anda.
3. Untuk akses yang lebih mudah, buat pintasan ke file `nahpu.exe` di desktop atau bilah tugas Anda.
4. Buka aplikasi dengan mengklik dua kali file `nahpu.exe` atau pintasan yang Anda buat.
5. Jika Anda menemukan peringatan tentang penerbit yang tidak dikenal, klik "Info lebih lanjut" lalu "Jalankan saja" untuk melanjutkan instalasi.

### Android

Persyaratan: Android 8.0 (Oreo) atau lebih baru.

Untuk menginstal aplikasi, ikuti langkah-langkah berikut:

1. Unduh versi terbaru aplikasi dari [GitHub](https://github.com/nahpu/nahpu/releases/latest/download/nahpu-Android.apk).

    - [APK Android Lama (32-bit)](https://github.com/nahpu/nahpu/releases/latest/download/nahpu-armeabi-v7a-release.apk).
    - [APK ChromeOS (64-bit)](https://github.com/nahpu/nahpu/releases/latest/download/nahpu-x86_64-release.apk). -->
2. Buka file APK yang diunduh di perangkat Android Anda.
3. Jika diminta, aktifkan instalasi dari sumber tidak dikenal di pengaturan perangkat Anda.
4. Ikuti petunjuk di layar untuk menyelesaikan instalasi.
5. Setelah terinstal, Anda dapat menemukan aplikasi di laci aplikasi atau layar beranda.

## Linux

Persyaratan: Ubuntu 22.04+ atau distribusi lain dengan GLIBC 2.34+

Untuk menginstal aplikasi, ikuti langkah-langkah berikut:

1. Unduh versi terbaru aplikasi dari [GitHub](https://github.com/nahpu/nahpu/releases/latest/download/nahpu-Linux-x86_64.tar.gz).
2. Ekstrak file `nahpu-Linux-x86_64.tar.gz` yang diunduh ke folder di komputer Anda.
3. Jalankan file yang dapat dieksekusi nahpu.

## Perangkat Apple (iOS/iPadOS/macOS)

Persyaratan:

- iOS 15.6 atau lebih baru
- iPadOS 15.6 atau lebih baru
- macOS 11.5 atau lebih baru

Apple mengharuskan penguji beta untuk menggunakan Apple [TestFlight](https://developer.apple.com/testflight/).

Dukungan perangkat:

- Daftar perangkat macOS tertua yang didukung dapat ditemukan di [Dukungan Apple](https://support.apple.com/en-us/102861).
- Dukungan perangkat untuk iOS dan iPadOS dapat ditemukan di [iosref](https://iosref.com/ios).

Untuk menginstal aplikasi, ikuti langkah-langkah berikut:

1. Unduh dan instal aplikasi TestFlight dari App Store di perangkat iOS atau iPadOS Anda.
2. Buka tautan ini di perangkat Anda dengan aplikasi TestFlight yang terinstal: [instal NAHPU](https://testflight.apple.com/join/DRG6yUfb).
