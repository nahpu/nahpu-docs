## Membuat proyek baru

Untuk membuat proyek baru, gunakan tombol `+` di sudut kanan bawah layar beranda. Anda akan diminta untuk memasukkan nama proyek, deskripsi proyek, dan format katalog utama.

- **Nama proyek** harus singkat dan deskriptif
- **Deskripsi proyek** harus berupa beberapa kalimat yang menjelaskan tujuan proyek.
- **Format katalog utama** adalah format katalog yang akan digunakan untuk menghasilkan halaman katalog spesimen. Anda dapat mengubahnya nanti. Anda juga dapat mencampur berbagai jenis format katalog dalam proyek yang sama.

## Mengenal dasbor

Setelah Anda membuat proyek, Anda akan melanjutkan ke dasbor proyek. Dasbor dibagi menjadi empat panel utama:

- **Tinjauan proyek** - Panel ini memberikan tinjauan singkat tentang proyek. Ini mencakup nama proyek, deskripsi, [peneliti utama](./projects#peneliti-utama), lokasi, tanggal mulai, tanggal berakhir, tanggal pembuatan basis data, dan tanggal akses terakhir.
- **Personel** - Panel ini mencantumkan semua personel yang terkait dengan proyek. Anda dapat menambahkan lebih banyak personel dengan mengklik tombol `[+ Tambah personel]` di bagian bawah panel.
- **Registri takson** - Panel ini mencantumkan semua taksa yang terdaftar di basis data. Anda dapat menambahkan lebih banyak taksa dengan mengklik tombol `[+ Tambah takson]` di bagian bawah panel. Anda juga dapat mengimpor taksa dari file csv dengan mengklik tombol `[Impor dari file]`.
- **Statistik catatan** - Grafik yang menunjukkan statistik waktu nyata dari jumlah catatan dalam proyek.

## Memperbarui info Proyek

Anda dapat menambahkan informasi lebih lanjut tentang proyek dengan mengklik tombol `pensil` di kanan bawah tinjauan proyek.

## Menghapus Proyek

Kami sengaja mempersulit penghapusan proyek. Jika Anda yakin ingin menghapus proyek, hapus semua catatan terkait dengan proyek terlebih dahulu dengan mengikuti semua langkah di bawah ini:

1. Buka dasbor proyek
2. Klik tombol narasi di bilah navigasi bawah. Gunakan tombol menu (ikon tiga titik) di sudut kanan atas untuk memilih `Hapus semua catatan`.
3. Ulangi langkah 2 untuk `Catatan Spesimen`, `Acara Pengumpulan`, dan `Catatan Situs`. Diperlukan urutan yang sama persis untuk menghapus catatan.
4. Kembali ke dasbor proyek dan hapus semua personel yang terdaftar.
5. Hapus proyek menggunakan tombol hapus di laci menu.

## Membuat Laporan

NAHPU menyediakan beberapa opsi untuk membuat laporan.

- **Jumlah spesies** - Ini akan menghasilkan laporan jumlah spesimen untuk setiap spesies dalam proyek.
- **Data Media** - Ini akan menghasilkan laporan semua file media dan metadatanya dalam proyek.

## Mengekspor Catatan

Opsi ekspor NAHPU:

- **Bundel catatan** - Opsi ini akan mengekspor semua catatan, termasuk media, dalam proyek sebagai file zip tunggal.
- **Ekspor catatan** - Opsi ini menyediakan ekspor untuk setiap jenis catatan dalam proyek. Anda memiliki opsi untuk mengekspor file Comma-delimited (.csv) atau Tab-delimited (.tsv).
- **Ekspor pdf** - Opsi ini akan mengekspor setiap jenis catatan sebagai file pdf.

## Terminologi

### Peneliti Utama

Peneliti utama adalah orang yang bertanggung jawab atas proyek tersebut.
