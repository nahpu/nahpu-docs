## Ringkasan

Anda dapat berkontribusi pada proyek dengan menulis dokumentasi. Beberapa cara untuk berkontribusi:

- **Perbarui dokumentasi yang ada**: Anda dapat membantu menjaga dokumentasi tetap terbaru dengan memperbarui halaman yang ada dengan informasi baru atau memperbaiki kesalahan.
- **Tambahkan dokumentasi baru**: Anda dapat membantu memperluas dokumentasi dengan menambahkan halaman atau bagian baru yang mencakup topik yang saat ini tidak dibahas.
- **Terjemahkan dokumentasi**: Anda dapat membantu membuat dokumentasi lebih mudah diakses dengan menerjemahkannya ke bahasa lain (segera hadir).

## Langkah-langkah Sederhana untuk Berkontribusi

Opsi ini untuk kontribusi cepat. Jika Anda ingin membuat perubahan yang lebih signifikan, silakan merujuk ke bagian "Langkah-langkah Lanjutan untuk Berkontribusi" di bawah ini.

### Anggota Tim

Di bagian bawah setiap halaman di situs web, Anda akan menemukan tautan "Edit halaman ini". Klik tautan ini untuk membuka halaman di GitHub. Anda kemudian dapat membuat perubahan langsung di GitHub dan mengirimkan permintaan tarik.

### Kontributor Eksternal

Jika Anda bukan anggota tim, Anda masih dapat berkontribusi pada proyek. Begini caranya:

1. Fork repositori.
2. Buat perubahan pada dokumentasi.
3. Kirimkan permintaan tarik.

## Langkah-langkah Lanjutan untuk Berkontribusi

Langkah ini memungkinkan lebih banyak fleksibilitas. Dokumentasi ditulis dalam Markdown dan dibuat menggunakan Docusaurus. Anda dapat mempelajari lebih lanjut tentang Docusaurus dengan mengunjungi [situs web resmi](https://docusaurus.io/).
