## Resumen

Puede contribuir al proyecto escribiendo documentación. Algunas formas de contribuir:

- **Actualizar la documentación existente**: Puede ayudar a mantener la documentación actualizada actualizando las páginas existentes con nueva información o corrigiendo errores.
- **Agregar nueva documentación**: Puede ayudar a ampliar la documentación agregando nuevas páginas o secciones que cubran temas que no se abordan actualmente.
- **Traducir documentación**: Puede ayudar a que la documentación sea más accesible traduciéndola a otros idiomas (próximamente).

## Pasos sencillos para contribuir

Esta opción es para contribuciones rápidas. Si desea realizar cambios más significativos, consulte la sección "Pasos avanzados para contribuir" a continuación.

### Miembros del equipo

En la parte inferior de cada página del sitio web, encontrará un enlace "Editar esta página". Haga clic en este enlace para abrir la página en GitHub. Luego puede realizar cambios directamente en GitHub y enviar una solicitud de extracción.

### Colaboradores externos

Si no es miembro del equipo, aún puede contribuir al proyecto. A continuación, le indicamos cómo:

1. Bifurque el repositorio.
2. Realice cambios en la documentación.
3. Envíe una solicitud de extracción.

## Pasos avanzados para contribuir

Este paso permite una mayor flexibilidad. La documentación está escrita en Markdown y construida con Docusaurus. Puede obtener más información sobre Docusaurus visitando el [sitio web oficial](https://docusaurus.io/).
