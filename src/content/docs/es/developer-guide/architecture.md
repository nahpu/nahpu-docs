## Resumen

NAHPU es una aplicación multiplataforma creada con [Flutter](https://flutter.dev/), un marco de alto rendimiento para crear aplicaciones multiplataforma. Flutter utiliza el lenguaje de programación Dart, que está optimizado para crear interfaces de usuario. Un enfoque con Flutter simplifica el proceso de desarrollo y permite a los desarrolladores crear aplicaciones para múltiples plataformas utilizando una única base de código. En el backend, NAHPU utiliza [SQLite](https://www.sqlite.org/index.html) para el almacenamiento local. Desde principios de 2024, hemos estado trabajando en agregar más código Rust a la base de código de NAHPU. Nuestro objetivo principal es reescribir las partes más lentas de la base de código en Rust. Rust es un lenguaje de programación de sistemas que se centra en el rendimiento, la fiabilidad y la seguridad. Al integrar Rust con Flutter, nuestro objetivo es mejorar el rendimiento de la aplicación y proporcionar una experiencia de usuario más sólida.
