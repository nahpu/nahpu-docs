Esta página contiene información sobre la generación de etiquetas de especímenes utilizando los datos de NAHPU.

## Ventajas de imprimir etiquetas de especímenes

- **Ahorra tiempo en el campo.** En lugar de escribir etiquetas completas de especímenes, lo que puede llevar fácilmente 10 o más minutos por etiqueta, una alternativa más rápida es usar etiquetas pequeñas con identificadores como un número de tejido (por ejemplo, “B-729113”) y un número de campo (por ejemplo, “JPO 7299”). Luego, de regreso en el museo, los usuarios pueden imprimir las etiquetas finales y completas para los especímenes.
- **Etiquetas más limpias y estandarizadas.** Algunas personas tienen el don de una caligrafía impecable, mientras que otras son menos afortunadas. ¡Imprimir las etiquetas asegura que todos los datos de la etiqueta sean fáciles de leer!
- **Evita errores derivados de tareas repetitivas y tediosas.** Escribir un nombre de localidad detallado con coordenadas y datos de elevación puede ser tedioso de escribir docenas de veces en el campo. El error humano es común. Al usar NAHPU, esta información solo necesita ingresarse una vez y luego puede asociarse con cada espécimen a través de un menú desplegable de sitios. Si es necesario, los detalles de la localidad también se pueden actualizar fácilmente dentro de la aplicación.
- **Permite una fase de limpieza de datos.** Antes de imprimir, es fácil usar una hoja de cálculo final para corregir coordenadas extrañas, elevaciones, errores de terminología, ortografía, etc.

## Flujo de trabajo de impresión de etiquetas

- **Exportar datos de especímenes de NAHPU como un archivo .csv.**
- **Limpieza final de datos.** Corrección ortográfica final, corrección de otros errores diversos, organización/combinación/división de columnas de datos según sea necesario.
- **Subir la hoja de cálculo a Entomolabels.** Con el [software Entomolabels](https://labels.entomo.pl/) (lamentablemente, solo disponible para computadoras con Windows) y plantillas de etiquetas personalizadas, puede dar formato a los datos en la etiqueta del espécimen.
- **Exportar los PDFs finales de las etiquetas para el frente y el dorso.** Combinar el PDF del frente de la etiqueta y el PDF del dorso, alternando las páginas frontal y posterior.
- **Usar una impresora con dúplex automático para imprimir las etiquetas finales.**

## Equipo

- **Impresora.** Para imprimir etiquetas, el equipo de NAHPU ha utilizado la [Epson WorkForce Pro WF-3820](https://epson.com/For-Work/Printers/Inkjet/WorkForce-Pro-WF-3820-Wireless-All-in-One-Printer/p/C11CJ07201) con [tinta pigmentada de secado instantáneo DURABrite® Ultra](https://www.amazon.com/gp/product/B08DX9DNSJ/ref=ppx_yo_dt_b_search_asin_title?ie=UTF8&th=1). Usando esta impresora, tinta y el papel que se muestra a continuación, imprimimos recientemente (primavera de 2024) más de 1000 etiquetas de especímenes de aves para el Museo de Ciencias Naturales de la Universidad Estatal de Luisiana (LSU).
- **Papel.** Hemos utilizado [papel 100% algodón, libre de ácido](https://www.redrivercatalog.com/browse/aurora-art-white-250.html) con un peso de 250 gsm. Dado que se trata de un papel relativamente grueso y resistente, descubrimos que era muy útil flexionar un poco el papel antes de imprimir. De lo contrario, la impresora puede tener problemas para dar la vuelta a la página internamente.
