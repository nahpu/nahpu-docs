El sistema NAHPU está diseñado para agilizar el proceso de curación y gestión de colecciones de historia natural, desde la recolección en el campo hasta la curación en el museo. Esta guía proporcionará algunos enfoques para este proceso. En última instancia, dependerá de la institución decidir la mejor manera de implementar el sistema para satisfacer sus necesidades.
