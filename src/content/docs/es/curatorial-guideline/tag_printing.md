Esta página contiene información sobre la generación de etiquetas de especímenes utilizando los datos de NAHPU.

## Ventajas de imprimir etiquetas de especímenes

- **Ahorra tiempo en el campo.** En lugar de escribir etiquetas de especímenes completas, lo que puede llevar fácilmente 10 minutos o más por etiqueta, una alternativa más rápida es usar etiquetas pequeñas con identificadores como un número de tejido (p. ej., "B-729113") y un número de campo (p. ej., "JPO 7299"). Luego, de vuelta en un museo, los usuarios pueden imprimir las etiquetas finales y completas para los especímenes.
- **Etiquetas más ordenadas y estandarizadas.** Algunas personas tienen el don de una escritura ordenada, mientras que otras tienen menos talento. ¡Imprimir etiquetas garantiza que todos los datos de la etiqueta sean fáciles de leer!
- **Evita errores de tareas repetitivas y tediosas.** Escribir un nombre de localidad detallado con coordenadas y datos de elevación puede ser tedioso de escribir docenas de veces en el campo. El error humano es común. Con NAHPU, esta información solo debe ingresarse una vez y luego puede asociarse con cada espécimen a través de un menú desplegable de sitios. Si es necesario, los detalles de la localidad también se pueden actualizar fácilmente dentro de la aplicación.
- **Permite la fase de limpieza de datos.** Antes de imprimir, es fácil usar una hoja de cálculo final para corregir coordenadas, elevaciones, errores de terminología, ortografía, etc.

## Flujo de trabajo de impresión de etiquetas

- **Exportar datos de especímenes de NAHPU como un archivo .csv**
- **Limpieza final de datos.** Corrección ortográfica final, corrección de otros errores misceláneos, organización/combinación/división de columnas de datos según sea necesario.
- **Cargar hoja de cálculo en Entomolabels.** Con el [software Entomolabels](https://labels.entomo.pl/) (desafortunadamente, solo disponible para computadoras con Windows) y plantillas de etiquetas personalizadas, puede formatear los datos en la etiqueta del espécimen.
- **Exportar pdf de etiquetas finales para el anverso y el reverso de las etiquetas.** Combine el pdf del anverso de la etiqueta y el pdf del reverso de la etiqueta, alternando las páginas del anverso y el reverso.
- **Use una impresora dúplex automática para imprimir las etiquetas finales.**

## Equipo

- **Impresora.** Para imprimir etiquetas, el equipo de NAHPU ha utilizado la [Epson WorkForce Pro WF-3820](https://epson.com/For-Work/Printers/Inkjet/WorkForce-Pro-WF-3820-Wireless-All-in-One-Printer/p/C11CJ07201) con [tinta de pigmento de secado instantáneo DURABrite® Ultra](https://www.amazon.com/gp/product/B08DX9DNSJ/ref=ppx_yo_dt_b_search_asin_title?ie=UTF8&th=1). Usando esta impresora, tinta y el papel que se muestra a continuación, imprimimos recientemente (primavera de 2024) más de 1000 etiquetas de especímenes de aves para el Museo de Ciencias Naturales de la Universidad Estatal de Luisiana.
- **Papel.** Hemos utilizado [papel 100% algodón, sin ácido](https://www.redrivercatalog.com/browse/aurora-art-white-250.html) con un gramaje de 250 g/m². Dado que este es un papel relativamente grueso y resistente, descubrimos que era muy útil flexionar un poco el papel antes de imprimir. De lo contrario, la impresora puede tener problemas para voltear la página internamente.
