**_Repensando los inventarios de biodiversidad en la era digital_**

NAHPU (**NA**tural **H**istory **P**roject **U**tility), nombrado libremente en honor al Gran Ciervo Ratón Malayo, [_Tragulus napu_](https://animaldiversity.org/accounts/Tragulus_napu/), es una aplicación de catalogación de campo diseñada para colecciones de historia natural. Es un proyecto gratuito y de código abierto con soporte para los principales sistemas operativos de escritorio y móviles. La versión actual está equipada para catalogar especímenes de mamíferos y aves.

:::info
NAHPU se encuentra actualmente en fase beta. Estamos trabajando para añadir más funciones y soporte para otros taxones. Si tienes alguna pregunta o estás interesado en contribuir al desarrollo de la aplicación, por favor contacta al [mantenedor del proyecto](https://hhandika.com/contact.html).
:::

## Características principales

- 📸 **Gestión de activos digitales**: Asocia activos digitales, como fotos de especímenes y hábitats, con tus registros de colección. Soporte para añadir fotos para sitios, especímenes y notas de campo.
- 🐦 **Soporte multi-taxón**: Puedes incorporar todos los taxones soportados en un solo proyecto, con cada entrada de espécimen teniendo un formato de catálogo adaptado específicamente para el taxón apropiado (por ejemplo, aves, mamíferos, murciélagos, etc.).
- 🌍 **Escribe el sitio una vez, úsalo en todas partes**: Crea un perfil de sitio con información detallada sobre la ubicación, el hábitat y un número ilimitado de coordenadas. Asócialo con múltiples especímenes, eventos de recolección y notas de campo.
- 📅 **Registra eventos de recolección**: Mantén un seguimiento de tus actividades de recolección documentando información detallada sobre la fecha de recolección, el método, los esfuerzos y el personal involucrado. Esto se puede asociar con múltiples especímenes.
- 🧬 **Gestión eficiente de muestras de tejido**: Gestiona un gran número de muestras de tejido de manera eficiente y mantén un seguimiento de su fecha y hora de recolección. El sistema admite la lectura de códigos de barras y códigos QR utilizando la cámara del dispositivo para un seguimiento fácil y preciso.
- 📊 **Estadísticas en tiempo real**: Supervisa las actividades, toma decisiones informadas para evitar la sobre-recolección y utiliza las estadísticas de preservación de tejidos para gestionar el muestreo en medio de recursos limitados de almacenamiento y preservación.
- 🔄 **Función de compartir avanzada**: Aprovecha la función de compartir avanzada de la aplicación para compartir tus datos con colaboradores o incorporarlos como una canalización automática a la base de datos del museo.
- 💪 **Impulsado por código abierto y estándares abiertos**: NAHPU está construido utilizando pilas de software de código abierto y se adhiere al estándar [Darwin Core](https://dwc.tdwg.org/). Sigue los [principios FAIR](https://www.go-fair.org/) para garantizar que los datos sean encontrables, accesibles, interoperables y reutilizables.

## La misión de NAHPU

### Simplificar el complejo trabajo de campo 🛠️

NAHPU está diseñado pensando en los estudios genómicos y el trabajo de campo remoto. Simplifica la asociación de activos digitales (por ejemplo, fotografías o grabaciones de audio) y partes de especímenes (por ejemplo, muestras de tejido o parásitos) con especímenes de bono. Hemos optimizado NAHPU utilizando lenguajes de programación y marcos de alto rendimiento para garantizar un funcionamiento fluido en varios dispositivos, al tiempo que se minimiza el uso de la batería. Nuestros esfuerzos continuos se centran en mejorar el rendimiento y la estabilidad de NAHPU.

### Apoyar la investigación de campo responsable y reproducible ❤️

NAHPU te permite registrar información detallada sobre los especímenes que recolectas, incluyendo origen, fecha de recolección, sitios, medidas, personal de campo, método de recolección y otra información relevante. Con estadísticas en tiempo real, NAHPU te ayuda a supervisar tus actividades de recolección y a tomar decisiones informadas para evitar la sobre-recolección. Juntas, estas características garantizan un trabajo de campo responsable y reproducible.

### Promover el intercambio de datos y la colaboración 🤝

NAHPU se desarrolla para agilizar los esfuerzos y fomentar la colaboración. La aplicación facilita el intercambio de datos y la colaboración entre investigadores al adherirse a estándares abiertos. Te permite crear una canalización automática para simplificar la transferencia de datos desde el trabajo de campo a la base de datos del museo. La versión actual es parcialmente compatible con el estándar [Darwin Core](https://dwc.tdwg.org/) y sigue los [principios FAIR](https://www.go-fair.org/). Nuestro objetivo es mejorar aún más el cumplimiento de NAHPU con estos estándares y principios.

### Accesible para todos ♿️

NAHPU es una aplicación gratuita y de código abierto que es compatible con las principales plataformas de escritorio y móviles. Está diseñada para coleccionistas e investigadores de historia natural con prioridad en la facilidad de uso y la accesibilidad. Aprovechamos el sistema [Material Design](https://m3.material.io/) para garantizar una interfaz intuitiva y un atractivo visual, al tiempo que se adapta a las personas con discapacidades. Nuestros esfuerzos continuos incluyen la internacionalización con soporte para múltiples idiomas y [configuraciones regionales](https://en.wikipedia.org/wiki/Locale_(computer_software)). Únete a nuestra diversa comunidad de colaboradores para hacer de NAHPU un proyecto inclusivo y acogedor para todos. ¡Siéntete libre de usar y contribuir a la aplicación!
